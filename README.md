# iTerm2 Color Schemes

A collection of **239** color schemes for [iTerm2](https://iterm2.com/), extracted from an iTerm2 preferences file.

## Usage

### Import a Color Scheme

1. Open **iTerm2** > **Settings** > **Profiles** > **Colors**
2. Click the **Color Presets** dropdown (bottom right)
3. Select **Import**
4. Navigate to the `themes/` folder and select the `.itermcolors` file(s) you want

Or simply **double-click** any `.itermcolors` file and iTerm2 will offer to import it.

### Regenerate Previews

Preview images are generated with Docker:

```sh
cd docker
./run.sh ../themes ../previews
```

The script builds the Docker image automatically if needed.
